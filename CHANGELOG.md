# Changelog

## [3.0.0] - 2025-01-04

### Initial Release

**Features:**
- Multi-framework support (QBCore, ESX, QBX)
- Multi-notification systems (6+ supported)
- Automatic vote detection (30s interval)
- Connection-based verification
- Manual vote claiming
- MySQL database integration
- Anti-exploit system (2h cooldown)
- Discord webhook with clickable vote link
- Multi-language (FR/EN)
- Admin commands
- Debug mode
- Sound effects & chat messages
- Automatic vote reminders
- Database auto-cleanup
- IP caching (5min)

**Technical:**
- Vote check: 30 seconds
- IP cache: 5 minutes  
- DB cleanup: 30 days
- Cooldown: 2 hours
